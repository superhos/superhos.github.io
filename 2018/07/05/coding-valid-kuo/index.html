423. 有效的括号序列

2018-07-05

Description

给定一个字符串所表示的括号序列，包含以下字符： ‘(‘, ‘)’, ‘{‘, ‘}’, ‘[‘ and ‘]’， 判定是否是有效的括号序列。

Example

括号必须依照 “()” 顺序表示， “()[]{}” 是有效的括号，但 “([)]”则是无效的括号。
O(n)的时间，n为括号的个数

Solution

这类题目也不用多想，用栈。两个栈比较。O(n).
