在Vue中使用JSX

Posted on 2019-10-14

虽然Vue的模板够简单，但是感觉上还是没有JSX用得顺手，这里来记录下Vue配置JSX的步骤。

什么是JSX

JSX本质上是语法糖，实际上是转化成了createElement函数。

使用步骤

  1. vue create 创建一个vue的项目

  2. 安装babel vue jsx编译库:

  3. 在.babelrc中添加下面代码：

    “presets”: [“@vue/babel-preset-jsx”]
